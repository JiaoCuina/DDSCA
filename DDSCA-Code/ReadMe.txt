Diagnosis-Guided Deep Subspace Clustering Association Study for Pathogenetic Markers
Identification of Alzheimer's Disease Based on Comparative Atlases
Author: Cui-Na Jiao, jiaocuina123@126.com

"DDSCA_Code" is a package written in Matlab. This repository contains the following files:

Main_DDSCA.m is the main function of the DDSCA package. 
f_SR.m calculates the the self-representation matrix. 
f_sCCALR.m is our proposed algorithm that compute association analysis at the outputs. 

deepnet is network toolbox
BrainNetwork is our brain network construction process
